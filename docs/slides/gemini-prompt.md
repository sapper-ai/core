### 슬라이드 7: 고도화 현황

**제목**: 지금 만들고 있는 것

**2컬럼 레이아웃** (각각 카드 형태):

**좌측 카드 (파란 테두리): MCP Security Proxy**
```
AI Client
  ↕ stdin/stdout
[SapperAI Proxy] ← Guard.preTool() / Guard.postTool()
  ↕
Real MCP Server
```
- ✅ 코드 변경 0줄
- ✅ 모든 도구 호출 자동 인터셉트
- ✅ 위험 시 JSON-RPC 에러로 차단

**우측 카드 (빨간 테두리): Adversary Campaign Runner**
- 5가지 내장 공격 시나리오
  - Prompt Injection
  - Command Injection
  - Data Exfiltration
  - Code Injection
  - Path Traversal
- 4가지 변형 자동 생성 (x4 = 20 테스트)
- → 탐지율 리포트 자동 출력

**스피커 노트**:
이미 동작하는 제품 위에 두 가지를 고도화하고 있습니다. MCP Proxy는 기존 MCP 서버를 코드 변경 없이 감쌉니다. 모든 도구 호출이 자동으로 보안 검증을 거칩니다. Adversary Campaign은 5가지 공격을 4가지 변형으로 자동 생성하여 탐지율을 검증합니다.
