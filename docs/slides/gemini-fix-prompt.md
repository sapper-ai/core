# SapperAI Deck 수정 요청

아래 3개 슬라이드를 수정해줘.

---

## 슬라이드 1 (타이틀) — 텍스트가 안 보임

현재 문제: 화면이 거의 완전히 검정색이라 아무것도 안 보임.

수정:
- 제목 "SapperAI"를 #84cc16 (올리브/라임 그린) 색상으로 크게 중앙에 배치
- 부제목 "AI 에이전트의 도구 호출을 실시간으로 보호합니다"를 #e5e5e5 (밝은 회색)으로
- 하단에 "김규동 | 조코딩 x OpenAI x Primer AI 해커톤"을 #9ca3af (회색)으로
- 방패 로고 아이콘은 제목 위에 올리브 색상으로
- 배경은 #0a0a0a 유지하되 텍스트가 확실히 보이게

---

## 슬라이드 3 (메트릭) — 숫자와 % 기호가 겹침

현재 문제: "96"과 "%"가 세로로 분리되어 겹쳐 보임. "0"과 "%"도 마찬가지. "0.002m"과 "s"도 분리됨.

수정:
- 첫 번째 카드: 숫자를 "96%" 한 줄로 합쳐서 크게 표시 (줄바꿈 없이)
- 두 번째 카드: "0%" 한 줄로 합쳐서 크게 표시 (줄바꿈 없이)
- 세 번째 카드: "0.002ms" 한 줄로 합쳐서 크게 표시 (줄바꿈 없이)
- 폰트 크기가 너무 크면 카드 안에서 줄바꿈이 발생하니, 카드 너비에 맞게 폰트를 조절해서 반드시 한 줄에 들어가게

## 슬라이드 6 (경쟁 포지셔닝) — 텍스트 잘림 + 색상 불일치

현재 문제:
1. 포지셔닝 맵의 축 레이블이 잘림: "Gatewa y", "Developer-Firs t", "Enterprise-Firs t"
2. 비교표 헤더가 파란색인데 전체 다크+올리브 테마와 안 맞음

수정:
1. 포지셔닝 맵 축 레이블을 줄바꿈 없이 한 줄로: "Gateway", "Developer-First", "Enterprise-First", "Tool Layer Focus"
2. 맵 크기를 약간 줄이거나 폰트를 줄여서 텍스트가 잘리지 않게
3. 비교표 헤더 색상을 파란색에서 #84cc16 (올리브) 배경 또는 #1a1a1a (다크 그레이) 배경으로 변경
4. SapperAI 열의 값들은 올리브 색상 유지
